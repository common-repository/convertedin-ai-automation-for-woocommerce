== Installation ==

1. login to your WordPress dashboard
2. open plugins tab > add new > upload plugin
3. Download and install the plugin
4. after installing the plugin successfully you'll be redirected to the screen where you can set the Convertedin email
5. Upon clicking `confirm and continue` the plugin will create an account on ConvertedIn platform using your WordPress user email or the email you chose. if you already have an account with the email you chose your store will be added to that email and your datasource is successfully added

 #### You can always login to your account using the WordPress Plugin

1. navigate to ConvertedIn tab in your WordPress Dashboard
2. Click `Login to ConvertedIn`
